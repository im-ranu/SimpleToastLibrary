# SimpleToastLibrary
Implementation Simple Toast in your Application


Step to integrate

Add it in your root build.gradle at the end of repositories:

allprojects {
		repositories {
			...
			maven { url 'https://jitpack.io' }
		}
	}
  
  
  Add the dependency
  
  dependencies {
	        implementation 'com.github.im-ranu:SimpleToastLibrary:Tag'
	}
